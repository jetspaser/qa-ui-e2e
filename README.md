# YouTrack UI experiments

Небольшой экспериментальный репозиторий для исследования и проверки различных подходов к UI-автотестированию YouTrack.

Проект используется как sandbox для:
- проверки гипотез по UI e2e тестам
- сравнения паттернов и анти-паттернов
- отработки конкретных сценариев, которые появляются по ходу задач

Репозиторий **не является законченным продуктом** и **не фиксирован по архитектуре**. Код будет постепенно обрастать новыми тестами и вспомогательными слоями по мере появления следующих заданий.

---

## Текущий фокус

На данный момент реализован базовый UI-сценарий:
- логин пользователя
- проверка успешной загрузки основного интерфейса

Данный сценарий используется как отправная точка для:
- обсуждения корректности UI-логина в e2e
- дальнейшего рефакторинга
- перехода к более устойчивым подходам (session reuse, storage state, API preconditions и т.д.)

---

## Что планируется дальше

По мере появления новых задач репозиторий будет расширяться, в том числе:
- выделение Page Object / App Actions
- переиспользование сессий без UI-логина
- добавление негативных сценариев
- стабилизация локаторов
- интеграция дополнительных проверок

Архитектурные решения могут меняться намеренно, как часть экспериментов и обсуждений.
